# 需求文档

## 1. 应用概述

### 应用名称
AI 知识库可视化产品

### 应用描述
面向 AI 产品经理的知识库可视化工具，通过交互式界面展示 AI PM 核心能力（Prototype、Benchmark、Eval），帮助用户快速理解和应用 AI 产品方法论。

---

## 2. 用户与使用场景

### 目标用户
- **主要用户**：AI 产品经理、产品经理、AI 研究员
- **次要用户**：技术团队、产品团队负责人

### 核心使用场景

1. **场景一：学习 AI PM 方法论**
   - 触发条件：新入行的 AI 产品经理需要快速了解核心能力
   - 用户目标：系统学习 Prototype、Benchmark、Eval 三个核心技能
   - 预期结果：通过可视化界面理解概念和应用场景

2. **场景二：快速验证产品想法**
   - 触发条件：产品经理有一个产品构想，需要快速验证
   - 用户目标：通过 Vibe Coding 快速生成原型
   - 预期结果：在 1-2 小时内获得可交互的原型

### 3.2 全局视觉与交互规范

#### 色彩体系
- 主色：`#667eea`（品牌紫）
- 辅助色：`#764ba2`（渐变紫）、`#52c41a`（成功绿）、`#faad14`（警告黄）、`#ff4d4f`（错误红）
- 中性色：`#333`（主文本）、`#666`（次文本）、`#999`（辅助文本）

#### 动画规范
- 默认缓动：`cubic-bezier(0.4, 0, 0.2, 1)`
- 快速动画：`150ms`
- 标准动画：`300ms`
- 慢速动画：`500ms`
- 降级方案：`prefers-reduced-motion: reduce` → `0.01ms`

#### 响应式断点
```css
/* 桌面端 - 默认 */
/* 平板端 */
@media (max-width: 900px) { /* 两列布局 */ }
/* 手机端 */
@media (max-width: 700px) { /* 单列布局 */ }
```

### 3.3 术语速查表详细说明

#### 内容
- 按分类展示术语：prototype, benchmark, eval, core, product, tool
- 每个术语包含：中文名、英文名、定义、举例

#### 布局
- 顶部：分类标签栏（横向滚动）
- 主体：术语卡片网格（响应式）
- 每张卡片：展示中文名 + 英文名

#### 样式

|------|----------|----------|

## 4. 业务规则与逻辑

### 规则 1：术语分类过滤
- **触发条件**：用户点击分类标签
- **处理逻辑**：
  1. 获取选中分类
  2. 过滤 TERMS 数据
  3. 动态渲染术语卡片
  4. 更新统计数量
- **输出结果**：仅显示选中分类的术语

### 规则 2：术语详情浮层定位
- **触发条件**：用户点击术语卡片
- **处理逻辑**：
  1. 获取卡片位置（rect）
  2. 计算浮层理想位置（卡片上方居中）
  3. 检测边界溢出
     - 如果超出顶部，翻转到卡片下方
     - 如果超出左右边界，水平偏移修正
  4. 应用定位
- **输出结果**：浮层完整可见，不超出 viewport

### 规则 3：数据驱动渲染
- **原则**：所有数据集中在 `data.js` 中管理
- **实现**：
  ```javascript
  const TERMS = {
    prototype: {
      zh: "原型",
      en: "Prototype",
      definition: "通过 Vibe Coding 缩短验证周期",
      examples: ["用 Claude 生成 Landing Page", "用 v0 生成组件"]
    },
    // ...
  };
  
  // 动态计算数量
  const termCount = Object.keys(TERMS).length;
  ```

## 5. 异常与边界情况

| 场景 | 异常类型 | 处理方式 | 用户提示 |
|------|----------|----------|----------|
| 数据缺失 | 空状态 | 显示占位图 | "暂无术语数据" |
| 浮层超出屏幕 | 边界溢出 | 自动翻转方向 | - |
| 分类无匹配 | 空结果 | 显示提示文案 | "该分类暂无术语" |
| 网络错误 | 请求失败 | 显示错误状态 | "加载失败，请重试" |
| 响应式断点 | 屏幕宽度 < 700px | 切换为单列布局 | - |
| 无障碍 | 键盘导航 | Tab 焦点可见 | - |
| 动画降级 | prefers-reduced-motion | 压缩至 0.01ms | - |

## 6. 验收标准

### 功能验收
- [ ] 用户可以通过点击分类标签过滤术语
- [ ] 用户可以通过点击术语卡片查看详情
- [ ] 浮层在不同位置均能完整显示
- [ ] 页面支持响应式布局（桌面/平板/手机）

- [ ] Chrome 最新版正常
- [ ] Safari 最新版正常

- [ ] 屏幕阅读器可读
- [ ] 颜色对比度 ≥ 4.5:1
- [ ] 动画支持 `prefers-reduced-motion`
