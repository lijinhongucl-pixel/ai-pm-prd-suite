# PRD 模板

本文档定义了生产级 PRD 的标准结构和撰写规范。

---

## 完整模板

```markdown
# 需求文档

## 1. 应用概述

### 应用名称
[产品名称]

### 应用描述
[一句话描述产品的核心价值和目标用户]

---

## 2. 用户与使用场景

### 目标用户
- **主要用户**：[用户画像描述]
- **次要用户**：[用户画像描述]

### 核心使用场景
1. **场景一**：[场景描述]
   - 触发条件：[什么情况下使用]
   - 用户目标：[用户想达成什么]
   - 预期结果：[产品如何满足]

2. **场景二**：[场景描述]
   - ...

---

## 3. 页面结构与功能说明

### 3.1 整体页面结构

```
页面名称
├── 顶部导航区 (Header)
│   ├── Logo
│   ├── 导航菜单
│   └── 用户操作区
├── 主内容区 (Main)
│   ├── 区块 A
│   │   ├── 子区块 A-1
│   │   └── 子区块 A-2
│   └── 区块 B
└── 底部信息区 (Footer)
    ├── 链接组
    └── 版权信息
```

### 3.2 全局视觉与交互规范

#### 色彩体系
- 主色：`#1890ff`（品牌蓝）
- 辅助色：`#52c41a`（成功绿）、`#faad14`（警告黄）、`#ff4d4f`（错误红）
- 中性色：`#333`（主文本）、`#666`（次文本）、`#999`（辅助文本）

#### 动画规范
- 默认缓动：`cubic-bezier(0.4, 0, 0.2, 1)`
- 快速动画：`150ms`
- 标准动画：`300ms`
- 慢速动画：`500ms`
- 降级方案：`prefers-reduced-motion: reduce` → `0.01ms`

#### 响应式断点
```css
/* 桌面端 - 默认 */
/* 平板端 */
@media (max-width: 900px) { ... }
/* 手机端 */
@media (max-width: 700px) { ... }
```

#### 内容
- [描述区块包含的内容]

#### 布局
- [描述区块的布局方式]

#### 样式

  /* CSS 样式定义 */

## 5. 异常与边界情况

| 场景 | 异常类型 | 处理方式 | 用户提示 |
|------|----------|----------|----------|
| 数据缺失 | 空状态 | 显示占位图 | "暂无数据" |
| 边界溢出 | 浮层超出屏幕 | 自动翻转方向 | - |
| 网络错误 | 请求失败 | 显示错误状态 | "网络异常，请重试" |
| 响应式断点 | 屏幕宽度 < 700px | 切换为单列布局 | - |
| 无障碍 | 键盘导航 | Tab 焦点可见 | - |

---

## 6. 验收标准

### 功能验收
- [ ] 用户可以通过 [操作] 完成 [目标]
- [ ] 系统在 [场景] 下正确显示 [结果]
- [ ] [功能名称] 符合 [预期行为]

### 性能验收
- [ ] 页面首屏加载时间 < 2s
- [ ] 交互响应时间 < 100ms
- [ ] 动画帧率 ≥ 60fps

### 兼容性验收
- [ ] Chrome 最新版正常
- [ ] Safari 最新版正常
- [ ] Firefox 最新版正常
- [ ] 移动端 Safari/Chrome 正常

### 无障碍验收
- [ ] 键盘可完全操作
- [ ] 屏幕阅读器可读
- [ ] 颜色对比度 ≥ 4.5:1

---

|------|------|----------|
| [功能名称] | [不实现的原因] | [后续排期] |
| ... | ... | ... |
```

---

## 撰写检查清单

### 必须包含
- [ ] 7 个完整模块
- [ ] 树状页面结构图
- [ ] CSS 类名预设
- [ ] 动画参数（类型、缓动、时长、延迟）
- [ ] `prefers-reduced-motion` 降级
- [ ] 响应式断点
- [ ] 异常情况表格
- [ ] 验收标准列表

### 禁止出现
- [ ] 硬编码数量（如"共 10 个术语"）
- [ ] 模糊描述（如"有动画"）
- [ ] 缺少单位（如"延迟 30"）
- [ ] 空白模块
